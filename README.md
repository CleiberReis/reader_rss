# readerrss

Um sumarizador de notícias, de diversas fontes, agrupadas por categorias.


# Requisitos

    pip install --user nltk tqdm feedparser beautifulsoup4
  
  
Rode um shell python, e execute o comando a seguir:

    >> import nltk
    >> nltk.download('punkt')
  

Após isso, rode o programa:

    python reader.py
